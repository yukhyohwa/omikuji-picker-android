# Omikuji Picker - Random Choice Helper

A professional, minimalist random-drawing application inspired by traditional Japanese fortune telling.

## Features
- **Traditional Interaction**: Real-target simulation with a shaking cylinder and physical lot-popping animation.
- **Modern Minimalist UI**: A premium, clean design focused on focus and elegance.
- **Full Customization**: 
  - Adjust the number of lots (1 to 24).
  - Customize the text for every individual lot.
- **Persistence**: All settings are automatically saved to your local device.

## How to Use
1.  **Launch**: Open the app to see the main cylinder.
2.  **Draw**: Tap the "SHAKE" button. The cylinder will oscillate, and a lot will pop up with your choice.
3.  **Configure**: Tap the gear icon in the bottom right to open the settings drawer.
4.  **Manage**: Change the count or edit labels, then tap "Apply Changes".

## Technical Profile
- **Platform**: Android (WebView-based Container)
- **Engine**: HTML5 / CSS3 (Flexbox/Transitions) / Vanilla JavaScript
- **State**: LocalStorage Persistence
